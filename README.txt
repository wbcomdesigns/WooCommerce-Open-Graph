== Description ==

**Enhanced Woo Open Graph** is the most comprehensive social media optimization plugin for WooCommerce stores. It automatically generates optimized Open Graph meta tags, Twitter Cards, and Schema.org structured data to ensure your products look perfect when shared on social media platforms.

= Key Features =

* Complete Social Media Support: Facebook, Twitter, LinkedIn, Pinterest, WhatsApp, and Email
* Schema.org Structured Data: Boost SEO with rich snippets and product markup
* Modern Social Share Buttons: Multiple styles (Modern, Classic, Minimal) with flexible positioning
* Advanced Image Handling: Multiple image sizes with smart fallback options
* Professional Admin Interface: Clean, responsive design with real-time validation
* Performance Optimized: Lightweight code that loads only when needed
* Developer Friendly: Extensive hooks, filters, and customization options

== Installation ==

= Automatic Installation =

1. Go to your WordPress admin panel
2. Navigate to **Plugins > Add New**
3. Search for "Enhanced Woo Open Graph"
4. Click **Install Now** and then **Activate**
5. Go to **WooCommerce > Open Graph** to configure settings

= Manual Installation =

1. Download the plugin ZIP file
2. Upload it via **Plugins > Add New > Upload Plugin**
3. Activate the plugin
4. Configure settings under **WooCommerce > Open Graph**

= Configuration =

1. **Enable Platforms**: Choose which social media platforms to support
2. **Set Image Preferences**: Select image sizes and fallback images
3. **Configure Social Sharing**: Choose button styles and positioning
4. **Advanced Settings**: Enable Schema markup and other features

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =

Yes, this plugin is specifically designed for WooCommerce stores and requires WooCommerce to be installed and active.

= Will this plugin slow down my website? =

No, the plugin is performance-optimized and only loads resources on product pages. The additional load time is typically less than 0.1 seconds.

= Can I customize the appearance of share buttons? =

Absolutely! The plugin includes three built-in styles (Modern, Classic, Minimal) and provides extensive CSS classes for custom styling.

= Does it work with other SEO plugins? =

Yes, the plugin can work alongside other SEO plugins. You can choose to override titles and descriptions from other plugins or let them work together.

= Is the plugin translation ready? =

Yes, the plugin is fully internationalized and includes POT files for easy translation. It also supports RTL languages.

= Can I add custom social media platforms? =

Yes, developers can add custom platforms using the provided hooks and filters. The plugin architecture is designed for extensibility.

= Does it support variable products? =

Yes, the plugin fully supports all WooCommerce product types including variable products, grouped products, and external products.

= Will my existing meta tags be affected? =

The plugin is designed to work harmoniously with existing meta tags. You can configure it to override other plugins' tags if needed.

= Is there analytics support? =

Yes, the plugin includes Google Analytics 4 integration and custom event triggers for comprehensive social sharing analytics.

= Can I use this with page builders? =

Yes, the plugin works with all major page builders including Elementor, Divi, Beaver Builder, and others.
