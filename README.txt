== Description ==

**Woo Open Graph** helps your WooCommerce products look perfect when shared on social media. This comprehensive plugin automatically generates optimized Open Graph meta tags, Twitter Cards, and Schema.org structured data to boost your social media presence and SEO.

== Installation ==

= Automatic Installation =

1. Go to your WordPress admin dashboard
2. Navigate to **Plugins → Add New**
3. Search for "Enhanced Woo Open Graph"
4. Click **Install Now** and then **Activate**
5. Go to **WooCommerce → Social Media** to configure settings

= Manual Installation =

1. Download the plugin ZIP file
2. Upload via **Plugins → Add New → Upload Plugin**
3. Activate the plugin through the **Plugins** menu
4. Configure settings under **WooCommerce → Social Media**

= Quick Configuration =

1. **Enable Platforms**: Select which social media platforms to support
2. **Choose Style**: Pick your preferred share button style
3. **Set Position**: Choose where share buttons appear on product pages
4. **Configure Images**: Set image sizes and fallback options
5. **Advanced Options**: Enable Schema markup and other features

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =

Yes, this plugin is specifically designed for WooCommerce stores and requires WooCommerce to be installed and active.

= Will this plugin slow down my website? =

No! The plugin is performance-optimized and only loads resources on product pages. The additional load time is typically less than 0.1 seconds.

= Can I customize the share button appearance? =

Absolutely! The plugin includes three built-in styles (Modern, Classic, Minimal) and provides extensive CSS classes for custom styling.

= Does it work with other SEO plugins? =

Yes, the plugin works harmoniously with other SEO plugins like Yoast, RankMath, and SEOPress. You can choose to override their meta tags if needed.

= Is the plugin translation ready? =

Yes, the plugin is fully internationalized with POT files included. It also supports RTL languages and follows WordPress translation standards.

= Can I add custom social media platforms? =

Yes, developers can add custom platforms using the provided hooks and filters. The plugin architecture is designed for extensibility.

= Does it support variable products? =

Yes, the plugin fully supports all WooCommerce product types including variable products, grouped products, and external products.

= Will my existing meta tags be affected? =

The plugin is designed to work alongside existing meta tags. You can configure it to override other plugins' tags or let them work together.

= Is there analytics support? =

Yes, the plugin includes Google Analytics 4 integration and custom event triggers for comprehensive social sharing analytics.

= Can I use this with page builders? =

Yes, the plugin works with all major page builders including Elementor, Divi, Beaver Builder, Gutenberg, and others.

= How do I test if it's working? =

Use these free testing tools:
* Facebook Sharing Debugger
* Twitter Card Validator  
* Google Rich Results Test
* LinkedIn Post Inspector

= Can I position share buttons anywhere? =

Yes! The plugin offers 4 built-in positions plus a `[wog_social_share]` shortcode for custom placement anywhere on your site.

= Does it work on mobile devices? =

Absolutely! All share buttons are optimized for mobile devices with touch-friendly interfaces and native sharing APIs where available.
